# manga.client

### clonning

```sh
git clone https://github.com/kuzanatoliy/manga.client.git
```

### install

For working you need nodeJS and yarn.

```sh
yarn install
```

### starting

```sh
yarn start
```

### building

```sh
yarn run build
```

### testing

```sh
yarn run test
```
